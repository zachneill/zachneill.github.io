BCVM

BCVM shows which vending machines at Berea College are in short supply. It is a web application written with an Angular front end and a Spring Boot Java back end. Hosted by the Heroku Saas, it connects to a PostgreSQL database and can be installed as a PWA. The app's data is crowd-sourced, but there are ReCAPTCHA security measures. Additionally, users may create accounts; their passwords are encrypted under modern hashing and salting cryptographic protocol.

SigBuild

SigBuild takes user inputs and creates a Berea College signature for the ends of emails. It is a web application coded in React.js. The app uses the open-source UI library MUI to assist with presentation.

Going Autopilot

Going Autopilot is a capstone project for the Digital Media minor. It is an in-browser game that depicts the main character undergoing the effects of amnesia but trying to live life normally. The game utilizes only the HTML, CSS, JavaScript, and JQuery library. It was built off the Twine prototyping tool and heavily modified.
